About
-----
RamCOH is a Python library for processing and peak fitting of Raman spectra, targeted at CO\ :sub:`2`\  fluids and (hydrous) silicate glasses

Installation
------------
ramCOH can be installed with pip by running:

.. code-block:: bash

    pip install ramCOH

in a terminal
